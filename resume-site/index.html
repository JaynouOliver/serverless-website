Serverless Website

07/2022

Serverless resume website with view counter using AWS. Built to participate in Forrest Brazeal's Cloud Resume Challenge.

  • Built a responsive site in HTML using the Bootstrap framework.
  • View counter uses a Lambda function written in GO lang to update DynamoDB's atomic counter.
  • The Lambda function runs when API is called and displayed using JavaScript.
  • View counter uses a function on the JSON response to bypass API Gateway's decimal encoding limitation.
  • Website front-end is deployed through CloudFront with an S3 origin.
  • Infrastructure is fully automated using CloudFormation and the AWS Serverless Application Model (SAM).
  • CI/CD pipeline using GitHub Actions with YAML.
